# Contributing to TerraTrace

First off, thanks for taking the time to contribute! 🎉

## Development Setup

1. Clone the repo and install dependencies with `npm install`
2. We use React 19, Vite, and Tailwind CSS v4.
3. Start the dev server with `npm run dev`.

## Pull Request Process

1. Ensure any install or build dependencies are removed before the end of the layer when doing a build.
2. Update the README.md with details of changes to the interface.
3. You may merge the Pull Request in once you have the sign-off of two other developers.

## Code Quality Standards
- We strictly enforce ESLint rules: `npm run lint:security`
- All files must be formatted with Prettier: `npm run format`
- Code must pass 100% of Vitest Unit and A11y tests: `npm run test:coverage`
